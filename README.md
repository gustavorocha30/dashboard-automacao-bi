Sistema de Monitoramento de Recomendações com Alertas Automatizados.

Tecnologias: Google Sheets, Google Apps Script (JavaScript), Power BI.

O Problema: A empresa tinha dificuldade em cobrar prazos e visualizar o status de pendências de forma clara.

A Solução: * Desenvolvimento de um banco de dados em Google Sheets.

Criação de um script em JavaScript (GAS) que varre a planilha semanalmente e envia e-mails personalizados para cada responsável com base no status das tarefas.

Dashboard no Power BI conectado à planilha para visualização executiva (gráficos de rosca e linha do tempo).
